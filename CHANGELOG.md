# Changelog

## 0.1.0-alpha2

- Исправлена остановка установки на хеше пароля с символами `$`.
- `web.env` больше не подключается как shell-скрипт.
- Проверка Ubuntu 22.04/24.04 выполняется до установки панели.
- AmneziaWG устанавливается до запуска web-панели.
- Сообщение `Ready` больше не появляется при незавершённой установке.

## 0.1.0-alpha1

- первый отдельный репозиторий только для AmneziaWG 2.0;
- полностью отделён от SG-Panel и Xray;
- отдельные пути, база, службы и установщик;
- настройка AWG 2.0, клиенты, `.conf`, QR, handshake и RX/TX;
- мониторинг общей памяти и load average;
- установка без Docker на Ubuntu 22.04/24.04.
