# Changelog

## 0.1.0-alpha6

- Типографика, ширина бокового меню, рабочие отступы, размеры карточек, полей, кнопок и таблиц приведены к визуальной системе SG-Panel.
- Основной фон сделан тёмно-серым; яркий зелёный акцент заменён на спокойный серо-синий. Зелёный используется только для успешных состояний.
- На странице Server добавлены кнопки автоматического определения публичного IPv4 и внешнего интерфейса.
- Добавлены проверенный профиль основных параметров маскировки и безопасная генерация новых диапазонов H1–H4 без смены ключей.
- Для каждого клиента добавлены отдельные название, комментарий, DNS и MTU. Пустые DNS и MTU наследуются от сервера.
- В Routing добавлен готовый режим «Только сеть AWG» наряду с полным туннелем и пользовательскими AllowedIPs.
- Добавлены миграции SQLite для индивидуальных DNS и MTU без потери существующих клиентов.
- Документация обновлена под новые настройки и порядок работы.

## 0.1.0-alpha5

- Увеличена читаемость интерфейса, добавлены Diagnostics, uptime и память процесса панели.
- Добавлены ежедневные резервные копии, защита входа и необязательная установка HTTPS.
- Python-пакет устанавливается в virtualenv, поэтому команда `awgpanel` работает из любого каталога.

## 0.1.0-alpha4

- Интерфейс разделён на Overview, Server, Clients, Access, Routing, DNS, Diagnostics, Backups, Security и Settings.
- Добавлены персональные ссылки на актуальный клиентский `.conf`, счётчик обращений и пересоздание токена.
- Добавлен выбор `AllowedIPs`, серверная изоляция клиентов, резервные копии и смена пароля.
- Исправлен лёгкий updater без `apt`, `dpkg` и перезапуска рабочего AWG.

## 0.1.0-alpha3

- Исправлена проверка временного файла через допустимое имя `awgtest0.conf`.
- Добавлены автоматическое определение публичного IPv4 и интерфейса, Diagnostics и память процесса панели.

## 0.1.0-alpha2

- Исправлена запись хеша пароля и ранняя проверка поддерживаемой версии Ubuntu.

## 0.1.0-alpha1

- Первый отдельный репозиторий только для AmneziaWG 2.0.
- Настройка AWG 2.0, клиенты, `.conf`, QR, handshake и RX/TX.
