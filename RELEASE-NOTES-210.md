# SG-AWG-Panel 210

210 продолжает рабочую основу 208 и возвращает классический интерфейс. Экспериментальный дизайн 209 в сборку не входит.

## Обновление поверх 208/209

```bash
unzip 210-AWG-Panel.zip
cd 210-AWG-Panel
sudo bash update.sh
```

Обновлятор сохраняет базу, клиентов, ключи, Cluster, Cascade, AWG-конфигурации и `web.env`. Перед копированием создаётся резервная копия. При ошибке код и данные автоматически возвращаются к предыдущему состоянию.

## Исправления

- одна видимая SG-Node на одно имя, без нескольких одинаковых блоков;
- команда подключения с токеном показывается сразу после добавления или повторного подключения;
- компактная таблица Clients без обязательной горизонтальной прокрутки;
- маршрут вида `Cascade → имя сервера выхода`;
- результат проверки Cascade непосредственно рядом с кнопкой;
- применение Direct/Cascade без ручного переподключения VPN;
- сохранены встроенные модальные окна, полный сброс внешнего Cascade, переименование сервера и SVG-флаги;
- WARP не добавлялся.
