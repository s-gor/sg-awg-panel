# SG-AWG-Panel 211

## Главное

- сохранён классический интерфейс 210;
- исправлен экспорт обычного клиента, созданного на SG-Node;
- добавлены самодостаточные `.run`-установщики для Controller и Node, которым не нужен `unzip`;
- установочный прогресс снова показывает зелёную вертушку, этап и время, а технический вывод уходит в журнал.

## Профили SG-Node

Удалённый клиент экспортируется в том же формате, что и локальный клиент Controller:

```ini
# Name = CC2-Node/Test-Client
# Client = Test-Client
# Source = SG-AWG-Panel
```

Особый комментарий `SG-AWG-Panel Cluster` и лишняя строка `# Server` больше не используются. Имя профиля берётся от реальной SG-Node. Для максимальной совместимости импортёров в профиль записывается один основной DNS-сервер. Endpoint использует подтверждённый runtime-порт Node.

## Установка на чистой Ubuntu

Основной формат новой установки:

```bash
chmod +x 211-INSTALL-SG-AWG-PANEL.run
sudo ./211-INSTALL-SG-AWG-PANEL.run
```

Для SG-Node:

```bash
chmod +x 211-INSTALL-SG-AWG-NODE.run
sudo ./211-INSTALL-SG-AWG-NODE.run
```

Оба файла содержат нужные исходники внутри и распаковываются через стандартный `tar`, который уже присутствует в Ubuntu Server. Предварительная установка `unzip` не требуется.
