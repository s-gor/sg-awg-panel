# SG-AWG-Panel v0.1.0 RC5 Hotfix 1

Промежуточная сборка перед отдельной технической доработкой Cascade.

## Исправлено и добавлено

- имя сервера запрашивается при чистой установке и редактируется в System;
- имя сервера и публичный IP/домен постоянно отображаются в верхней шапке;
- версия панели и UI-сборки отображаются в нижнем блоке бокового меню;
- меню приведено к порядку System → Cluster → Cascade → Clients → AWG Server → Network → Security → Maintenance → Help;
- в клиентский `.conf` добавлено имя клиента и имя сервера;
- добавлен QR-код полного AmneziaWG-конфига;
- добавлена персональная обновляемая подписка `.conf`;
- добавлен управляемый JSON-профиль для интеграции с SG Client;
- улучшено визуальное подтверждение копирования ссылок и подписок;
- исправлена ширина рабочих карточек, длинных URL, конфигураций и правой колонки;
- узкие окна безопасно переводят многоколоночные блоки в одну колонку;
- сохранены циферблаты памяти и системного диска.

## Граница этой сборки

Логика Cascade не изменялась. Эта сборка предназначена для чистой установки и проверки всех исправлений, которые должны быть готовы до отдельного ремонта и живого теста Cascade на двух EC2.
