# SG-AWG-Panel v0.1.0 RC5

RC5 переносит в SG-AWG-Panel общую архитектуру и рабочие сценарии SG-Panel RC70, адаптированные под AmneziaWG.

## Главное

- отдельный раздел **Cascade**: Exit-клиент, импорт на Entry, назначение клиентов, проверка и безопасное отключение;
- отдельный раздел **Cluster**: Controller, SG-Node, одноразовое подключение, heartbeat, метрики, службы и безопасные задания;
- создание первого AWG-профиля на SG-Node и выдача клиентского `.conf` после подтверждённого применения;
- полная встроенная **Help** с поиском и пошаговыми сценариями;
- единый `Current JSON` с обязательной многоуровневой проверкой;
- read-only **System Files** вместо неоднозначного `Generated Config`;
- установка файлов напрямую из архива 4PDA без загрузки исходников с GitHub.

## Безопасность SG-Node

- Controller не открывает SSH и не передаёт произвольные команды;
- enrollment token действует 30 минут, постоянный agent token хранится с правами `0600`;
- Agent принимает только фиксированные типы заданий;
- `PreUp`, `PostUp`, `PreDown`, `PostDown` и `SaveConfig` в управляемой AWG-конфигурации запрещены;
- конфигурации проверяются через `awg-quick strip` и `nft -c -f`;
- при ошибке применения восстанавливаются прежние файлы и службы;
- серверный приватный ключ очищается из задания после завершения.

## Проверки кандидата

Исходники проходят автоматические тесты, Python compile и синтаксическую проверку shell-скриптов. Перед публичной публикацией необходим реальный прогон на Controller, отдельной SG-Node и двух серверах Cascade.
