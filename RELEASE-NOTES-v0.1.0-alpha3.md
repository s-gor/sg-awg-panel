# SG-AWG-Panel v0.1.0 Alpha 3

Alpha 3 фиксирует все ошибки, найденные при первом реальном подключении AWG 2.0 на Ubuntu 24.04, и превращает рабочий прототип в более цельную тестовую панель.

## Главное

- исправлено имя временного файла для проверки `awg-quick`;
- установщик ждёт освобождения `apt/dpkg` после фоновых обновлений Ubuntu;
- публичный IPv4 и внешний интерфейс определяются автоматически;
- одна кнопка сохраняет, проверяет и запускает/перезапускает сервер;
- при ошибке возвращаются предыдущая база, `awg0.conf` и состояние службы;
- резервная копия создаётся перед каждым изменением сервера или клиента;
- добавлено пересоздание ключей клиента;
- добавлена отдельная страница Diagnostics;
- показана реальная RSS-память web-панели отдельно от общей памяти Linux;
- добавлены журналы, проверка UDP-порта, интерфейса и последние резервные копии;
- добавлены отдельные скрипты обновления и удаления;
- переписана документация по установке, клиентам AmneziaVPN и диагностике.

## Проверено до сборки

- 13 automated tests passed;
- Python syntax passed;
- Bash syntax passed;
- локальные Markdown-ссылки проверены;
- исполняемые права Bash-скриптов сохранены.

Сетевая часть Alpha 3 основана на реально проверенной Alpha 2: AWG 2.0 поднят на Ubuntu 24.04, клиент AmneziaVPN подключился, сайты открылись через публичный IP AWS, а панель показала handshake и RX/TX.

Alpha 3 после сборки ещё должна пройти обновление на действующем тестовом EC2 и затем чистую установку.
