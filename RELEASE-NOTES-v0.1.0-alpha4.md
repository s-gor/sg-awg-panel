# SG-AWG-Panel v0.1.0 Alpha 4

Alpha 4 развивает уже проверенный на реальном EC2 сервер AmneziaWG 2.0 и не меняет рабочую сетевую основу `awg0`.

## Главное

- новый интерфейс в стиле SG-Panel с отдельным бирюзовым цветом;
- чистый Overview без повторяющихся кнопок настройки;
- отдельные страницы Server, Clients, Access, Routing, DNS, Diagnostics, Backups, Security и Settings;
- персональные ссылки на текущий `.conf` клиента;
- счётчик скачиваний и пересоздание токена Access;
- полный или выборочный `AllowedIPs` для каждого клиента;
- изоляция клиентов друг от друга;
- отдельное управление DNS;
- ручные резервные копии и восстановление из панели;
- смена пароля администратора;
- подробные отдельные страницы документации.

## Исправление обновления

Updater теперь обновляет только код панели и Python-зависимости. Он:

- не запускает `apt update`;
- не использует `dpkg`;
- не ждёт постоянный процесс `unattended-upgrade-shutdown`;
- не переустанавливает AmneziaWG;
- не перезапускает работающий AWG-туннель.

Установщики ждут только реальные lock-файлы `apt/dpkg`.

## Совместимость

База Alpha 3 мигрирует автоматически. Сохраняются:

- настройки Server;
- серверные ключи;
- все клиенты и их ключи;
- адреса клиентов;
- рабочий `awg0.conf`.

## Ограничения

- Ubuntu 22.04/24.04;
- IPv4;
- один интерфейс `awg0`;
- web-панель пока по HTTP `:8080`;
- доменные и гео-правила пока не добавлены;
- Docker Compose будет отдельным способом установки позднее.
