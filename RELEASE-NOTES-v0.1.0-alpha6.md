# SG-AWG-Panel v0.1.0 Alpha 6

Alpha 6 приводит интерфейс к визуальной системе SG-Panel и добавляет основные настройки, которых не хватало для повседневной работы.

## Интерфейс

- те же размеры бокового меню, рабочие отступы, заголовки, карточки, поля, кнопки и таблицы, что в SG-Panel;
- тёмно-серый фон вместо почти чёрного;
- спокойный серо-синий акцент вместо яркого зелёного;
- зелёный оставлен только для успешных состояний.

## Server

- повторное автоматическое определение публичного IPv4;
- повторное автоматическое определение внешнего интерфейса;
- проверенный профиль основных параметров маскировки;
- генерация новых непересекающихся H1–H4 без смены ключей.

## Clients и Routing

- отдельные название и комментарий клиента;
- индивидуальные DNS и MTU с наследованием серверных значений;
- новый режим «Только сеть AWG»;
- сохранены полный туннель и пользовательские AllowedIPs;
- миграция существующей базы без потери клиентов и ключей.

## Проверки

- 31 automated tests passed;
- Python syntax passed;
- Bash syntax passed;
- all main pages and client editor rendered;
- editable package installation passed;
- Markdown local links passed;
- ZIP integrity passed.

Alpha 6 ещё не проверена на реальном EC2. Обновление не должно перезапускать рабочий AWG-туннель.
