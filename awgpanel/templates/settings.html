Публичный доступ

Python backend всегда слушает только 127.0.0.1:18080. Снаружи панель публикует Nginx на отдельном выбираемом порту.

Рекомендуемый порт SG-AWG-Panel: 62443. Порты 80 и 443 панель не занимает.

Оставьте включённым на отдельном сервере. Если TCP 443 и сертификат этого домена уже обслуживает другой Nginx-проект, выключите и используйте существующий сертификат.

Для первого выпуска сертификата откройте TCP 80. Для панели откройте выбранный TCP-порт. На TCP 443 будет только обычная заглушка.
