# Access

Для каждого клиента создаётся персональная ссылка вида:

```text
http://SERVER_IP:8080/a/SECRET_TOKEN
```

Она отдаёт текущий `.conf` этого клиента.

Это не VLESS-подписка. Автоматическое обновление зависит от клиентского приложения. Ссылка полезна, чтобы повторно скачать актуальный конфиг после изменения DNS или Routing.

Доступные действия:

- включить или отключить ссылку;
- создать новый токен;
- увидеть число обращений;
- увидеть время последнего скачивания.

Пока панель работает по HTTP, разрешайте TCP 8080 только со своего IP и не передавайте ссылку посторонним.
