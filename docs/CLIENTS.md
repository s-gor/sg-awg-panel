# Клиенты

Каждому устройству нужен отдельный клиент.

## Создание

1. Откройте **Клиенты**.
2. Укажите название устройства и, при необходимости, комментарий.
3. Нажмите **«Создать клиента»**.
4. Откройте **«Конфиг / QR»**.

## Индивидуальные настройки

Кнопка **«Настроить»** позволяет изменить:

- название устройства;
- комментарий;
- DNS только для этого клиента;
- MTU только для этого клиента.

Пустые DNS и MTU наследуются от страницы **Server**. После изменения DNS, MTU или Routing скачайте `.conf` заново.

## Импорт

- Windows: скачайте `.conf` и импортируйте в AmneziaVPN.
- Телефон: отсканируйте QR в приложении с поддержкой AWG 2.0.

Длинный текст вручную копировать не требуется.

## Действия

- **Отключить** — peer удаляется из серверного конфига.
- **Включить** — peer возвращается.
- **Пересоздать ключи** — адрес сохраняется, старый конфиг перестаёт работать.
- **Удалить** — клиент и ключи удаляются.

После подключения в таблице появляются handshake и RX/TX.
