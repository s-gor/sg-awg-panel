# JSON и Generated Config

## JSON

JSON-редакторы дают доступ к тем же данным, которые изменяются обычными формами.

Используйте их только при понимании структуры конфигурации.

Перед сохранением:

1. проверьте синтаксис JSON;
2. проверьте IP, сети и порты;
3. убедитесь, что не удалены необходимые поля;
4. сохраните резервную копию;
5. после применения проверьте AWG Server, Traffic Rules и клиента.

## Generated Config

`Generated Config` используется только для просмотра конфигурационных файлов, созданных панелью.

Не редактируйте эти файлы вручную: при следующем применении настроек панель может создать их заново.

Traffic Rules в RC4 поддерживают блокировку трафика.
