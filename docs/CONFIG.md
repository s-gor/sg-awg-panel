# JSON и System Files

## Current JSON

`Current JSON` — основной экспертный инструмент SG-AWG-Panel. Он содержит текущее сохраняемое состояние панели и доступен для редактирования.

Перед применением панель обязательно проверяет:

- синтаксис JSON с точной позицией ошибки;
- обязательные поля, типы и допустимые значения;
- ссылки на клиентов и Outbounds;
- IP, CIDR, домены, конфликты и дубликаты;
- временную генерацию runtime-конфигураций;
- `awg-quick strip` и `nft -c -f`.

Кнопка «Сохранить и применить» остаётся недоступной, пока именно текущий текст не прошёл проверку.

Контекстные JSON-редакторы находятся также в соответствующих разделах Clients, AWG Server, Network, Security и Maintenance.

## System Files

`System Files` показывает не исходные настройки, а фактические файлы, которые панель уже создала и использует:

- конфигурацию AWG Server;
- конфигурацию Nginx;
- Traffic Rules / nftables;
- управляемые systemd units.

Раздел предназначен только для чтения, копирования и скачивания. Ручное редактирование этих файлов может быть перезаписано панелью и не рекомендуется.
