# DNS

Страница DNS меняет строку `DNS = ...` в клиентских конфигурациях.

Поддерживается от одного до четырёх IP-адресов через запятую.

Готовые варианты:

```text
Cloudflare: 1.1.1.1, 1.0.0.1
Google:     8.8.8.8, 8.8.4.4
Quad9:      9.9.9.9, 149.112.112.112
```

Уже импортированный клиентский конфиг автоматически не меняется. После сохранения DNS скачайте `.conf` заново.
