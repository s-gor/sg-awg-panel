# Встроенная Help

Раздел **Help** доступен из главного меню панели и содержит поиск по всей справке.

Внутри описаны:

- установка и первый вход;
- System и диагностика;
- Clients;
- AWG Server;
- Network, Outbounds, Traffic Rules и JSON;
- Cascade, отключение, замена Exit и повторная проверка;
- Cluster, установка SG-Node, подключение, повторная регистрация и журналы;
- Security и HTTPS;
- Maintenance, backup, restore и update;
- список данных, которые нельзя публиковать в отчёте об ошибке.

Встроенная справка является основной пользовательской инструкцией и должна обновляться вместе с интерфейсом.
