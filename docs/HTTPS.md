# HTTPS

## Подготовка

1. Создайте DNS-запись `A`.
2. Направьте домен на публичный IP сервера.
3. Разрешите TCP `80`, TCP `443` и выбранный TCP-порт панели.

## Включение

1. Откройте `Security → Panel Access`.
2. Выберите `HTTPS + Let's Encrypt`.
3. Введите домен без протокола.
4. Проверьте порт панели.
5. Нажмите **Сохранить и применить**.

Панель будет доступна по адресу:

```text
https://YOUR_DOMAIN:ВЫБРАННЫЙ_ПОРТ
```

Порт `443` может использоваться страницей-заглушкой. Административная панель остаётся на выбранном отдельном порту.
