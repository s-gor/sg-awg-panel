# Установка SG-AWG-Panel

## Требования

- Ubuntu Server 22.04 LTS или 24.04 LTS;
- архитектура amd64;
- отдельный сервер или EC2;
- root или sudo;
- открытый UDP-порт AmneziaWG;
- отдельный TCP-порт панели.

## Установка из архива 4PDA

Скачайте прикреплённый архив, загрузите его на сервер и выполните:

```bash
unzip 208-AWG-Panel.zip
cd 208-AWG-Panel
sudo bash install.sh
```

Файлы панели устанавливаются непосредственно из архива. Доступ к GitHub для их загрузки не требуется.

Интернет на сервере необходим для установки системных компонентов и зависимостей.

Тот же установщик поддерживает повторную установку на этом Ubuntu после полного удаления и перезагрузки:

```bash
sudo reboot
# после повторного подключения распакуйте новый архив и снова выполните:
sudo bash install.sh
```

Действующая установка не перезаписывается. Автоматически удаляются только безопасные остатки уже удалённой панели.

## Установка из GitHub

```bash
curl -fsSL https://raw.githubusercontent.com/s-gor/sg-awg-panel/v210/install.sh | sudo bash
```

Укажите имя сервера, пароль администратора и публичный TCP-порт панели.

Рекомендуемый порт:

```text
62443
```

После завершения откройте:

```text
http://PUBLIC_IP:ВЫБРАННЫЙ_ПОРТ
```

Журнал установки:

```text
/var/log/sg-awg-panel-install.log
```

Установщик рассчитан на отдельный сервер. Не используйте полный uninstall на машине, где Nginx, Certbot или AmneziaWG обслуживают другие проекты.
