# Maintenance

## Резервные копии

Панель позволяет создавать и восстанавливать резервные копии.

Перед восстановлением:

1. убедитесь, что выбран нужный архив;
2. сохраните текущие клиентские конфигурации;
3. после восстановления проверьте панель, AWG Server и реального клиента.

## Обновление

Обновление можно запустить в `Maintenance → Updates`.

Ручной запуск:

```bash
sudo env SG_AWG_PANEL_VERSION=v0.1.0-rc4   bash /opt/sg-awg-panel/deploy/update-from-github.sh
```

После обновления:

1. выполните `Ctrl+F5`;
2. проверьте версию;
3. проверьте `sg-awg-panel.service`;
4. проверьте `sg-awg-server.service`;
5. проверьте Nginx;
6. подключите реального клиента.

RC4 не заявляет автоматический откат как гарантированную подтверждённую функцию.
