# Карта панели

Обычный порядок работы:

```text
Server → Clients → Access → Routing → DNS
```

- **Обзор** — текущее состояние без технических настроек.
- **Server** — endpoint, UDP-порт, сеть, MTU и параметры AWG 2.0.
- **Клиенты** — создание устройств, ключи, QR, `.conf`, handshake и трафик.
- **Доступ** — персональные ссылки на актуальный `.conf`.
- **Routing** — изоляция клиентов и `AllowedIPs`.
- **DNS** — DNS, который записывается в клиентские конфиги.
- **Diagnostics** — службы, uptime, автозапуск, UDP-порт, NAT, reboot-проверка, журналы и безопасный отчёт.
- **Резервные копии** — ежедневный timer, ручное создание и восстановление состояния.
- **Безопасность** — пароль, защита входа и текущий режим доступа.
- **Настройки** — HTTPS, домен, отдельный порт панели, заглушка 443, версия и обновления.
