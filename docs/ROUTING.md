# Routing

## Изоляция клиентов

Рекомендуемый режим блокирует прямой обмен между клиентами внутри `awg0`, но сохраняет их выход в интернет.

Технически панель добавляет firewall-правило для трафика:

```text
awg0 → awg0 → DROP
```

## AllowedIPs клиента

### Весь трафик

```text
0.0.0.0/0
```

Весь IPv4-трафик клиента идёт через VPS.

### Только указанные сети

Пример:

```text
10.0.0.0/8, 192.168.1.0/24
```

Через AWG пойдут только указанные сети.

После изменения `AllowedIPs` скачайте и импортируйте клиентский `.conf` заново. Доменные и гео-правила в Alpha 4 ещё не реализованы.
