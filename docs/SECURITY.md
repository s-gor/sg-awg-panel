# Security

## Пароль

Пароль администратора задаётся при установке и может быть изменён в панели.

После смены пароля активные сессии завершаются.

## IP allowlist

Можно разрешить доступ только с указанных IP-адресов или сетей CIDR.

Пустой allowlist разрешает доступ с любого IP.

Перед сохранением убедитесь, что текущий IP не будет заблокирован.

## Сессии

Панель показывает активные сессии и позволяет завершить отдельную сессию или все остальные.

## Журнал входов

Фиксируются:

- успешные входы;
- ошибочные попытки;
- временно заблокированные попытки;
- IP и User-Agent.

## Общие рекомендации

- используйте HTTPS;
- ограничьте SSH своим IP;
- не публикуйте клиентские конфигурации;
- не передавайте защищённые ссылки посторонним;
- храните пароль в менеджере паролей.
