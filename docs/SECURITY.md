# Безопасность Alpha 1

- TCP 8080 открывайте только для своего публичного IP.
- Используйте отдельный длинный пароль панели.
- Клиентские конфигурации содержат закрытые ключи.
- Не публикуйте `/var/lib/sg-awg-panel/panel.db` и `/etc/amnezia/amneziawg/awg0.conf`.
- На тестовом сервере не размещайте другие важные сервисы.

HTTPS и allowlist внутри панели будут добавлены после проверки базовой работы AWG.
