# Безопасность Alpha 3

- TCP 8080 открывайте только для своего публичного IP.
- Используйте отдельный длинный пароль панели.
- UDP 585 можно открыть для адресов, с которых будут подключаться клиенты, либо для всего интернета, если адреса меняются.
- Не публикуйте базу `panel.db`, файл `awg0.conf`, клиентские `.conf` и QR-коды.
- Один клиентский конфиг используйте только на одном устройстве.
- При потере файла клиента используйте **«Пересоздать ключи»**.
- Размещайте Alpha-версию только на отдельном тестовом VPS.

HTTPS и дополнительное ограничение доступа внутри панели планируются в следующих версиях.
