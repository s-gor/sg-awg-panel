# Безопасность

- TCP 8080 разрешайте только для своего публичного IP, пока не настроен HTTPS.
- Используйте длинный отдельный пароль панели.
- Пароль можно сменить на странице **Безопасность**.
- После пяти неудачных входов с одного IP новые попытки блокируются на оставшуюся часть 15-минутного окна.
- UDP 585 открывайте для AWG-клиентов.
- Не публикуйте `panel.db`, `awg0.conf`, клиентские `.conf`, QR-коды и ссылки Access.
- Один клиентский конфиг используйте на одном устройстве.
- При потере конфига пересоздайте ключи клиента.
- При утечке ссылки Access создайте новый токен.

Для постоянного сервера настройте HTTPS по инструкции [HTTPS.md](HTTPS.md). После этого закройте внешний TCP 8080.
