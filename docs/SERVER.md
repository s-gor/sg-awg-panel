# AWG Server

Раздел управляет основной конфигурацией AmneziaWG.

## Основные параметры

- публичный IP или домен Endpoint;
- UDP-порт;
- внутренняя сеть клиентов;
- DNS;
- MTU;
- внешний интерфейс;
- параметры AmneziaWG.

## Управление службой

Панель позволяет:

- запустить AWG Server;
- остановить AWG Server;
- перезапустить AWG Server;
- применить изменённые параметры.

После изменения Endpoint, UDP-порта, сети или параметров клиента обновите клиентские конфигурации и проверьте подключение реального устройства.

Traffic Rules для блокировки находятся в отдельном разделе `Network`.
