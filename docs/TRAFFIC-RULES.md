# Traffic Rules

Traffic Rules в версии `v0.7.0-RC5` используются для блокировки трафика клиентов AmneziaWG.

## Что можно блокировать

В правиле можно указать:

- всех клиентов или выбранных клиентов;
- один или несколько доменов;
- IPv4-адреса и сети CIDR;
- TCP или UDP;
- отдельные порты и диапазоны портов.

Правила выполняются по порядку сверху вниз.

## Создание правила

1. Откройте `Network → Traffic Rules`.
2. Нажмите **Добавить правило**.
3. Укажите понятное название.
4. Выберите клиентов или оставьте правило для всех клиентов.
5. Укажите домены либо IPv4-адреса и сети.
6. При необходимости выберите протокол и порты.
7. Выберите действие **Block**.
8. Сохраните и примените правило.
9. Проверьте результат на клиентском устройстве.

## Осторожное применение

Сначала создавайте узкое правило для одного тестового клиента. После проверки расширяйте его на остальных клиентов.

Правило для всех назначений может полностью лишить выбранного клиента доступа в интернет.
