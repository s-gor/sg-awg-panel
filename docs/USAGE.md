# Использование

## Настройка сервера

На главной странице заполните:

- **Публичный IP или домен** — IPv4 нового EC2;
- **UDP-порт** — `585`;
- **Сеть клиентов** — `10.77.0.0/24`;
- **DNS** — `1.1.1.1, 1.0.0.1`;
- **Внешний интерфейс** — обычно `ens5` на EC2;
- **MTU** — `1280`.

Параметры `H1–H4` можно оставить пустыми при первой настройке: панель создаст непересекающиеся диапазоны автоматически.

После сохранения появится файл:

```text
/etc/amnezia/amneziawg/awg0.conf
```

Нажмите **Запустить**. Статус должен стать `active`.

## Создание клиента

1. Введите название устройства.
2. Нажмите **Создать клиента**.
3. Откройте **Конфиг / QR**.
4. Скачайте `.conf` или отсканируйте QR-код клиентом с поддержкой AmneziaWG 2.0.

Один конфиг предназначен только для одного устройства.

## Проверка

На клиенте включите туннель и откройте любой сайт проверки IP. Затем на сервере:

```bash
sudo awg show awg0
```

В панели должны появиться handshake и счётчики RX/TX.

## Нагрузка

Внизу главной страницы показаны:

- load average за 1, 5 и 15 минут;
- использование RAM всего сервера.

Для точного измерения дополнительно используйте:

```bash
systemd-cgtop
systemctl status sg-awg-panel sg-awg-server
free -h
```
