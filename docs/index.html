SG-AWG-PANEL / CASCADE

Cascade

Понятный маршрут Client → Entry → Exit → Internet без ручной сборки правил

СХЕМА МАРШРУТА

Client → Entry → Exit → Internet

Entry принимает подключение клиента, Exit предоставляет внешний адрес. Отключение Cascade безопасно возвращает назначенных клиентов на обычный AWG-Gateway.

НЕ НАСТРОЕН
1 · ExitСоздать служебный доступПолучить одноразовую конфигурацию для Entry
2 · EntryИмпортировать и проверитьПроверка туннеля, маршрута и выходного IP
3 · КлиентыНазначить и включитьВыбранные клиенты направляются через Exit

Мастер Cascade

Кнопка включения становится доступной только после успешной проверки служебного туннеля.

1
Создайте служебного клиента на ExitКонфигурацию можно повторно открыть, пока служебный peer существует; кнопка явно подтверждает копирование.
2
Импортируйте конфигурацию на EntryПанель создаёт отдельный системный outbound SG-AWG Cascade.
3
Запустите полную проверкуПроверяются интерфейс, маршрут и фактический выход через выбранный outbound.
4
Назначьте клиентовМожно выбрать отдельных клиентов или все активные подключения.
