# Скриншоты SG-AWG-Panel 0.7.0-RC5

Рекомендуемый размер: `1600×900` или другой единый формат 16:9.

Подготовленные имена:

1. `01-system.png` — System: версия, имя сервера, ресурсы и службы.
2. `02-clients.png` — Clients: локальный и удалённый клиент, маршрут и трафик.
3. `03-cluster.png` — Cluster: Controller и одна SG-Node со статусом «В сети».
4. `04-cascade.png` — Cascade: `CC1 → CC2 → internet` и успешная проверка.
5. `05-ssh-menu.png` — команда `sudo sg-awg-panel` и нумерованное меню.
6. `06-installer.png` — этап установки с зелёной вертушкой.

Перед публикацией скрыть или заменить:

- enrollment token;
- PrivateKey и PresharedKey;
- пароли;
- персональные UUID;
- реальные имена клиентов;
- IP и домены, если их не нужно публиковать.

Не добавляйте в кадр вкладки браузера, рабочий стол и лишние терминальные команды.
